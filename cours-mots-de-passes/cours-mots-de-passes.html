Sécurité renforcée

Double protection

Même si votre mot de passe est volé, l'attaquant ne peut pas se connecter sans votre téléphone

Les differentes methodes 2FA

SMS

Code envoye par message texte

Pratique mais vulnerable aux attaques SIM swap

Email

Code envoye par email

Depend de la securite de votre boite mail

TOTP (code temporaire)

Code a 6 chiffres qui change toutes les 30s

Genere par une app authenticator - recommande !

Application Push

Notification sur votre telephone

Un tap pour approuver - tres pratique

Cle physique

YubiKey, Google Titan, etc.

Le plus securise - impossible a phisher

Codes de secours

Codes a usage unique a conserver

Indispensable en cas de perte du telephone !
