01 / Définition Cours

C'est quoi un plugin ?

"Les plugins sont des paquets de code qui permettent d'étendre et de personnaliser les fonctionnalités de WordPress."

Un plugin est écrit en PHP et peut contenir des images, du CSS ou du JavaScript.

Un plugin minimal nécessite un seul fichier PHP avec un en-tête spécifique appelé DocBlock.

Cas d'usage

🔌

Connecter une API externe

Afficher des données depuis un service tiers

🗄️

Accéder à une base de données

Lire/écrire des données métier personnalisées

⚙️

Ajouter des fonctionnalités

Formulaires, galeries, SEO, sécurité...

📂 Emplacement

Tous les plugins résident dans wp-content/plugins/

02 / Création Cours

Créer son premier plugin

Créer un plugin WordPress est très simple : un dossier, un fichier PHP, un commentaire.

5 étapes

1Accéder au dossier wp-content

4Créer un fichier PHP dedans

03 / Hooks Cours

Les Actions

Les hooks sont des points d'ancrage placés dans WordPress pour intervenir sans modifier les fichiers de base.

Les actions permettent d'appeler une fonction au moment où WordPress exécute une fonctionnalité spécifique.

add_action( $tag, $callback )

$tag

Nom du crochet (ex : 'wp_head', 'delete_post')

$callback

Fonction à exécuter quand l'action se déclenche

💡 Bonne pratique

Préfixer les noms de fonctions avec le nom du plugin pour éviter les conflits.

Exemple 1 — Ajouter une balise meta

// Ajouter une meta dans <head>
function mon_plugin_meta() {
  echo '<meta name="author"
        content="ESIG" />';
}

add_action('wp_head',
             'mon_plugin_meta');

Exemple 2 — Email à la suppression d'un article

function mon_plugin_notif($post_id) {
  wp_mail(
    'admin@esig.ch',
    'Article supprimé',
    'ID: ' . $post_id
  );
}

add_action('delete_post',
             'mon_plugin_notif');

Le paramètre $post_id est passé automatiquement par WordPress.

04 / Hooks Cours

Les Filtres

Les filtres permettent de modifier une valeur durant l'exécution de WordPress sans toucher à l'original.

add_filter( $tag, $callback )

Même syntaxe que add_action(), mais la fonction doit retourner la valeur modifiée.

🎯

Action

Exécute du code

🔄

Filtre

Modifie une valeur

⚠️ Attention

Un filtre doit toujours retourner la valeur, sinon le contenu disparaît !

Exemple — Remplacer "et" par "&" dans les titres

function mon_plugin_titre($title) {
  return str_replace(
    ' et ',
    ' & ',
    $title
  );
}

add_filter('the_title',
            'mon_plugin_titre');

Quelques filtres utiles

the_titleTitre de l'article
the_contentContenu de l'article
the_excerptExtrait de l'article
wp_titleTitre de la page
05 / Shortcodes Cours

Créer un shortcode

WordPress interdit l'ajout de PHP directement dans les articles pour des raisons de sécurité.

Les shortcodes sont des macros qui permettent d'injecter du contenu dynamique dans les pages et articles.

Syntaxe

Comme des balises HTML, mais avec des crochets [ ] au lieu de chevrons < >

HTML<img src="photo.jpg" />
WP[gallerie id="127" col="3"]

add_shortcode( $tag, $callback )

$tag

Nom du shortcode (ex : 'yoda')

$callback

Fonction qui retourne le HTML à afficher

Exemple — Shortcode [yoda]

function mon_plugin_yoda() {
  return '<blockquote>
    Que la force soit avec toi
    jeune padawan !
  </blockquote>';
}

add_shortcode('yoda',
               'mon_plugin_yoda');

Utilisation et résultat

Dans l'éditeur :

Bonjour à tous, voici une citation :

[yoda]

Résultat :

Bonjour à tous, voici une citation :

Que la force soit avec toi jeune padawan !
06 / Shortcodes avancés Cours

Enclosing shortcodes et attributs

Les enclosing shortcodes entourent du contenu pour le traiter, comme les balises HTML.

3 paramètres du callback

$atts

Tableau des attributs du shortcode

$content

Contenu entre les balises ouvrante et fermante

$tag

Nom du shortcode utilisé

Attributs personnalisés

Les shortcodes acceptent des paramètres :

[vador] → blockquote (défaut)
[vador tag="div"] → div

Exemple — Shortcode [vador]

function mon_plugin_vador(
  $atts, $content, $tag
) {
  $atts = shortcode_atts([
    'tag' => 'blockquote'
  ], $atts);

  return "<{$atts['tag']}>
    {$content},
    Je suis ton père !
  </{$atts['tag']}>";
}

add_shortcode('vador',
               'mon_plugin_vador');

Utilisation et résultat

[vador]Jean-Marc[/vador]
Jean-Marc, Je suis ton père !

💡 do_shortcode()

Permet d'exécuter un shortcode directement en PHP, utile pour imbriquer des shortcodes.

Shortcode [ico]

Créer un shortcode qui permet d'insérer rapidement des icônes PNG dans une page ou un article.

Cahier des charges

Récupérer le nom de l'image saisi entre les balises et ajouter .png

08 / Base de données Cours

Shortcodes et BDD

Les shortcodes permettent d'intégrer des données provenant de bases de données, d'API externes ou de fichiers de données.

Structure du plugin gest-cli

wp-content/plugins/
└── gest-cli/
    ├── gest-cli.php  ← principal
    ├── config.php    ← connexion
    ├── db.php        ← PDO
    └── client.php    ← métier

config.php

define('DB_HOST', 'localhost');
define('DB_NAME', 'ma_base');
define('DB_CHARSET', 'utf8');
define('DB_USER', 'user');
define('DB_PASS', '****');

⚠️ Sécurité

Toujours valider, nettoyer et échapper les données pour prévenir les injections SQL.

db.php — Connexion PDO

function getDB() {
  global $db;
  if ($db === null) {
    $db = new PDO(
      "mysql:host=".DB_HOST.
      ";dbname=".DB_NAME.
      ";charset=".DB_CHARSET,
      DB_USER, DB_PASS
    );
  }
  return $db;
}
